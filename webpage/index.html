Taxi driver agent

Panagiotis Giannakopoulos

Problem description

The given problem is to design a taxi driver agent who will be able to pick up the passenger at one location and drop him off in another. The simulator is provided by Open AI gym Taxi v3. The state space of the problem is described by 500 states, while the action space includes 6 possible actions. The project is developed in Python3.

The problem is solved with the application of Reinforcement Learning algorithms. Specifically, with the use of Q learning, SARSA learning and Deep Q Networks. In order to improve the performance of Q and SARSA, grid search is applied on the hyperparameters. The results of the algorithms are compared with goal to find the best algorithm.
