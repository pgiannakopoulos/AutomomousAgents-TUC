Taxi driver agent

Panagiotis Giannakopoulos

Problem description

The given problem is to design a taxi driver who can pick up the passenger at one location and drop him off in another. In this project, the simulation of this problem is provided by Taxi v2 Open AI simulator and it is solved using Reinforcement Learning algorithms. The project is developed in Python.
